Sommaire

  • Ce module ajoute une fonction de cache à votre site qui peut réduire considérablement la charge lorsque votre site a un nombre important d'accès.
  • La fonction de ce module est applicable à deux types de groupes d'utilisateur (les utilisateurs anonymes et les groupes d'utilisateur enregistrés).
    Le contenu en cache des utilisateurs n'est plus accessible à d'autres au même moment.

Gestion du cache en utilisant le bloc vider le cache

  • Le bloc vider le cache limite l'accès au contenu en cache quand le bloc est regardé par un utilisateur anonyme.
  • Nous recommandons de rendre le bloc accessible aux utilisateurs anonymes, et derégler le tempas de cache à "Un jour".
  • Le module pourrait créer une grande quantité de fichiers de cache. Mais ce n'est pas un problème si le bloc vider le cache est paramétré comme recommandé.

Comment configurer

  • Veuillez ajouter les deux lignes suivantes au fichier de configuration, sous le bloc [Cube].
  • CacheSystem.path=/modules/stdCache/kernel
  • CacheSystem. classe = StdRenderCache
