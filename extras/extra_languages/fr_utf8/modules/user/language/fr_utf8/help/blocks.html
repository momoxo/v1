Module Utilisateur - Blocs disponibles

Connexion
Sommaire

Ce bloc affiche le formulaire de connexion uniquement aux utilisateurs anonymes. Si l'enregistrement est activé, le lien de register.php est également affiché. Et dans le cas de l'authentification par SSL le lien spécifié dans les options des préférences.

Note

Vous pouvez vous connecter depuis user.php si par mégard vous avez désintallez ce bloc.

Qui est en ligne
Aperçu général

Ce bloc affiche le nombre et les nom des utilisateurs connectés sur votre site.

Note

Afficher l'information juste des utilisateurs connectés peut s'avérer une lourde charge. Pour cette raison, ce bloc ne cherche pas à corriger les résultats. Mais reste suffisant pour créer une interaction sur votre site.

Nouveaux Membres
Aperçu général

Comme le nom l'indique, ce bloc affiche les derniers utilisateurs enregistrés.

Options
Afficher x membres
Indique le nombre limite d'utilisateurs à afficher.
Afficher les avatars des membres
En cas d'OUI, affiche les avatars avec le nom de l'utilisateur.
Top Membres
Aperçu général

Ce bloc affiche les utilisateurs avec le plus grand nombre de contributions.

Options
Afficher x membres
Indique le nombre de limite d'utilisateurs à afficher.
Afficher les avatars des membres
En cas d'OUI, affiche les avatars avec le nom de l'utilisateur
Ne pas afficher les utilisateurs dont le Rang est x
Vous pouvez retirer des utilisateurs d'un groupes spécifique selon leur Rang.
