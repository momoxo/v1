Module Utilisateur

Sommaire

Le module Utilisateur a pour fonction générale l'administration des utilisateurs et des groupes.
Il possède les fonctions pour éditer les "Membres" et les "Groupes" du système de configuration de XOOPS 2.0
En utilisant le module 'Utilisateur', vous pouvez éditer les comptes des utilisateurs, changer la permission des groupes, modifier le rang d'un utilisateur, permettre ou rejeter l'enregistrement de nouveaux utilisateurs, envoyer un courrier par email ou message privé aux utilisateurs et ainsi de suite.

Installation
Vous n'avez pas besoin de faire grande chose dans la plupart des cas, parce que ce module est certainement déjà installé correctement. Si vous avez besoin de plus d'informations, consultez le "guide d'installation".
