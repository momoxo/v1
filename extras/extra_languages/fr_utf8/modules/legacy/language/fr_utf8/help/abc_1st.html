La première étape - L'interface d'administration et les Thèmes

Description

Nous allons voir en détail l'interface d'administration.
Ensuite, comment configurer l'information de votre site web depuis les Paramètres Généraux.
Et pour finir, comment changer le Thème.

La mise en page de l'interface d'administration
Menu Latéral

Vous allez devoir utiliser divers menus pour gérer votre site.
Sur la colonne à gauche vous voyez les menus de chaque module avec leurs fonctions.

Lors de l'étape d'installation, vous avez déjà installé quelques modules.
Les menus s'affichent sur la colonne de gauche. Le Menu du module courant est ouvert.
En attendant, les menus des autres modules restent fermés.
Pour ouvrir les menus de ces modules, cliquez sur l'icon [+].
Vous pouvez cliquer sur le titre des modules pour accéder à la page d'accueil de ces modules.


Chercher une Fonction

Vous trouverez la boîte de recherche d'une fonction sur le coin supérieur droit de la page.
C'est un dispositif de recherche spécial pour les administrateurs.
Vous pouvez rechercher une fonction d'un menu à partir de mots-clés.
Dans le cas où vous ne trouvez pas la fonction recherché, essayez ceci!

Par exemple, si vous avez oublié comment créer un nouveau utilisateur, essayez : 'créer utilisateur'.
La recherche d'une fonction affichera les résultats trouvés.

Autres Liens

Avec la boite de recherche d'une fonction vous trouverez également d'autres liens dans un menu:

Panneau de contrôle
Vous permet d'aller à la page d'accueil de l'interface d'administration.
Page d'accueil
Vous permet de retourner à la page d'accueil de votre site public sans vous déconnecter.
Déconnexion
Vous permet de vous déconnecter et retourner à votre page d'accueil public.
La première étape de votre Site (1) Paramètres Généraux

Nous allons commencer à configurer votre Site.
Tout d'abord, regardez le titre de votre nouveau Site sur la barre de votre navigateur.
' Cube Site - Just Use it'

Voici comment remplacer l'information des sites Officiels de Cube par la votre.
Vous pouvez changer les informations basiques comme le titre et le slogan dans les options des 'Paramètres Généraux'.
Cette fonction se trouve dans le menu du module Legacy.

Vous trouverez une trentaine d'options dans les 'Paramètres Généraux'.

Et voici les plus importantes options :

Nom du Site

Inscrivez ici le nom de votre Site web.

Slogan du Site

Idéal pour un sous-titre. Ajoutez un slogan évocateur a votre site web.

Fuseau horaire du serveur

L'installateur à certainement ajouté le fuseau horaire de votre serveur.
Veuillez cependant confirmer que l'information est correcte.
Si vous ignorez le fuseau horaire de votre serveur, laissez la valeur par défaut.

Fuseau horaire par défaut

Cette valeur est utilisé comme fuseau horaire par défaut lors de l'enregistrement d'un nouveau utilisateur.
Vous pouvez choisir le fuseau horaire de votre lieu de résidence ou celui de votre public.
Mais comme les utilisateurs peuvent changer leur fuseau horaire, ce n'est pas très important.
Si vous ignorez le fuseau horaire de votre public, laissez la valeur par défaut.

Mode de mise au point (déboguer)

Cube Legacy a trois types différents de mode de débogage qui permettent aux propriétaires du site de résoudre rapidement des problèmes. L'installateur a configuré 'Mise au Point PHP' comme mode par défaut. Il est conseillé de garder le mode 'Mise au Point PHP' pendant la configuration de votre site web. Mais, changez ceci vers le mode 'Inactif' lorsque votre site sera accessible au public.

Fermer votre site ?

Vous pouvez choisir de fermer votre site au public pendant que vous la configuration. Les administrateurs ont accés par défaut au site fermé. Cependant vous pouvez accorder des droits d'accés à d'autres groupes d'utilisateurs.

Valider!

Cliquez sur le bouton 'Valider' pour confirmer les changements dans la configuration de votre site.

La première étape de votre Site (2) - Préférences Utilisateur

Cube Legacy a pour objectif d'être un portail communautaire. L'enregistrement de nouveaux utilisateurs est possible dés la fin de l'installation. Vous pouvez empêcher tout enregistrement avec le module Utilisateur. Cliquez sur l'icone [+] ou le titre du module utilisateur, ensuite cliquez sur 'Préférences'.

Vous pouvez voir diverses options de configuration, comme dans l'image précédente, sur chaque menu de module.

Autoriser l'enregistrement de nouveaux utilisateurs

En attendant de configurer votre Site, cochez l'option 'Non' pour arrêter de nouveaux enregistrements. Et cliquez sur le bouton 'Valider'.

La première étape de votre Site (3) - Changez le Thème de votre Site

Cliquez sur le lien 'Page d'accueil' dans le coin supérieur droit de votre page. Vous voyez que l'apparence standard de votre Site est la même que cube.org... Cube utilise des Thèmes pour personnaliser l'apparence du Site. Vous pouvez changer l'image de votre Site en changeant le Thème. Un logiciel client FTP est essentiel pour placer sur le serveur de nouveaux Thèmes téléchargés, comme vous l'avez fait pour placer Cube.

Télécharger de Nouveaux Thèmes

Vous pouvez utiliser les thèmes de 2 avec Cube Legacy. Beaucoup de designers publient des themes de qualité sur leurs sites. Visitez xoopscube.org et Wiki pour obtennir les adresses URL, et télécharger vos themes favoris. Vous aurez également besoin d'utilitaires de décompression, parce que la plupart des themes sont compressés comme les archives de XOOPS Cube Legacy. Si vous utilisez WindowsXP ou Mac OS X, il vous sera très facile de décompresser les fichiers archives des thèmes.

Placer les Thèmes sur votre Site

Une fois votre Thème décompressé, placez le dossier dans le répertoire /themes de votre XOOPS Cube Legacy. Le dossier de votre Thème doit contennir un fichier theme.html et, ou package.ini.php. Veuillez à respecter l'arborescence suivante.

Attention! Une erreur très fréquente des utilisateurs, consiste à placer un dossier supplémentaire comme suit:

Selectionner votre Thème principal

Cliquez sur la fonction 'Themes' du module Legacy.

Vous pouvez choisir l'apparence de votre Site en cliquant sur le bouton 'Seléctionner' à coté de votre Thème favorit.

Conclusions

Vous avez terminez la première étape de votre Site. Votre site possède à présent son propre nom, slogan et apparence. Et vous êtes préparé pour l'étape suivant. Bien évidemment, vous pourrez changer cette configuration à tout moment.
