La deuxième étape - L'installation de blocs

Descripton

Vous pouvez ajouter un message de bienvenue pour amis et visiteurs dans l'en-tête de la page d'accueil. Mais également des blocs d'articles et forums selon les besoins de votre site.

La Mise en Page

Voici une mise en page d'un theme standard de Xoops Cube Legacy

Le site xoopscube.org utilise aussi les blocs dans la mise en page. C'est une structure basique de XOOPS.

Dans la plupart des thèmes, vous pouvez utiliser trois colonnes: la colonne de gauche, la colonne du centre et la colonne de droite.
Au centre, vous avez le plus souvent un bloc central et deux colonnes: centre-gauche, centre-centre et centre-droit.
Vous pouvez assigner un "Bloc" à une de ces colonnes.

Le "Bloc" est une partie de contenu dynamique.
Les blocs appartiennent aux modules et sont installés en même temps que les modules.

Bloc de connexion

Le bloc que vous utilisez pour vous connecter est le bloc "Login" (connexion) installé par défaut.
Ce bloc affiche les boîtes de texte pour le nom de l'utilisateur et le mot de passe.
Lors de la connexion, il bloc est automatiquement remplacé par le bloc 'Menu Utilisateur'.

Bloc du Menu Principal

Ceci est le bloc automatique du Menu Principal pour tous les utilisateurs.
Le bloc du Menu Principal affiche uniquement les lien accessibles à l'utilisateur actuel.

Le Bloc du Menu Utilisateur

C'est un bloc qui affiche un menu personnel pour les utilisateurs inscrits.
Ce bloc est aussi un de trois blocs installés par défaut.

La Deuxième Etape (1) - Comment installer de nouveaux blocs
Installer un Bloc

Commencez par installer le bloc "Site Info" (information du site) sur la colonne droite.
Ce bloc affiche les administrateur de votre site.
Ainsi, les visiteurs peuvent obtenir le surnom et l'adresse de contact de l'administrateur.
Cliquez sur "Installer un Bloc" du module Legacy dans le bloc gauche du panneau de contrôle.

Ensuite, vous obtenez la liste des blocs non-utilisés.

Trouvez le bloc "Site Info", et cliquez sur l'icône installer de la ligne indiquant ce bloc.

Installation du Bloc

Le bloc "Site Info" n'est pas encore installé. Vous devez configurer quelques options.

Titre

Le titre de ce bloc pour les visiteurs.

Position / Colonne

Choisissez une colonne pour afficher votre bloc. Cette fois, spécifiez "Bloc Droit".

Ordre

Ordre d'affichage sur la colonne. Vous pouvez laisser la valeur par défaut.

Temps de Cache

Vous pouvez spécifier combien de temps le système garde en cache ce bloc. Dans le cas où le cache est activé, le système ne fait pas appel au programme mais utilise le bloc sauvegardé dans le dossier cache.

Cibler des modules

Spécifier les modules dans lesquels vous voulez afficher le bloc. Cette fois, spécifiez "Toppage" (page d'accueil seulement).

Cibler un groupe

Spécifiez les groupes d'utilisateurs qui auront le droit d'accès au bloc. C'est quoi un groupe ?
Les groupes sont un ensemble d'utilisateurs avec la permission d'accéder à certains modules ou le panneau de contrôle.
Vous avez trois types de groupes du système. Ces groupes sont spéciaux, donc l'administrateur ne peut pas les effacer.

Webmestres
Ceci est le premier groupe de ce site.
Les utilisateurs dans ce groupe ont la permission d'exécuter toute action.
Le compte administrateur que vous utilisez lors de votre connexion appartient à ce groupe.
Utilisateurs inscrits
Les utilisateurs communs qui se sont enregistrés, appartiennent par défaut à ce groupe .
Utilisateurs anonymes
Ce groupe représente les utilisateurs non enregistrés.

Bien, cette fois, spécifier les trois groupes.
Parce que vous n'avez pas besoin de cacher le bloc aux utilisateurs.

Options

Cette option s'affiche dans le cas où un bloc a des paramètres facultatifs.
Mais, Cette fois, vous n'avez pas à le faire.

Installer

Une fois votre bloc configuré, cliquez sur le bouton "Installer" !
Allez sur la page d'accueil pour confirmer l'installation du bloc.

La Deuxième Etape (2)- Comment contrôler l'ordre des blocs

Installons un bloc supplémentaire, le bloc "Chercher" pour que les utilisateurs puissent effectuer une recherche avec un mot clé dans le contenu du site. Lors de l'installation de ce bloc, choisissez la position "Bloc Droit" et "Tous les modules".
A présent, la colonne droite a deux blocs: le bloc "Chercher" en haut, sur toutes les pages, et le bloc "Site Info" seulement en haut de la première page.

Pour changer l'ordre d'affichage (poids) de ces blocs, cliquez sur le lien "Gestion des Blocs" du module Legacy dans le bloc gauche du panneau de contrôle.

Vous voyez la liste des blocs installés.

Ajoutez la valeur 1 dans la colonne 'Ordre' (ou poids, i.e. xoops2) du bloc "Chercher". Et la valeur 5 sur le bloc "Site Info".

Ensuite, cliquez sur le bouton "Valider". Maintenant, le bloc "Chercher" est toujours affiché avant le bloc "Site Info".

La Deuxième Etape (2)- Comment utiliser des blocs personnalisés

Les blocs personnalisés sont aussi très utiles, par exemple, pour ajouter un message de bienvenue sur la première page de votre site.

Cliquez sur "Gestion de Blocs".
Ensuite, cliquez sur "Ajouter un Bloc Personnalisé".

Vous voyez le formulaire d'édition pour créer un nouveau bloc personnalisé.
Les champs de ce formulaire vous sont familiers et n'ont pas besoin d'explications.

Contenu

Le type de contenu du bloc détermine comment il sera traité lors du rendu.
Cette fois, entrez votre message de bienvenue.

Type

Ce champ détermine comment rendre le contenu du bloc. Il y a quatre types.

HTML
Le Type HTML vous permet d'ajouter du code html dans le champ 'contenu'.
PHP
Le Type PHP vous permet d'ajouter votre script dans le champ 'contenu'.
Le format auto avec émoticones
Ceci est le type le plus pratique. Vous pouvez entrer des messages dans le champ 'contenu' comme pour les articles ou le forum.
Le format auto sans émoticones
Vous ne pouvez pas utiliser les émoticones.

Cette fois, utilisez le format "Auto avec émoticones" et entrez votre message de bienvenue.
Et, assignez ce bloc à la colonne du centre accessible par tous les groupes.

Conclusions

Fondamentalement, les propriétaires du site n'utilisent pas d'éditeur HTML avec XOOPS Cube Legacy.
Certains thèmes vous offrent une mise en page et des colonnes différentes. Donc, vous pouvez assigner des blocs à ces colonnes spécifiques.
