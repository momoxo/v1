A faire pour la prochaine version de Cube Legacy 2.1.0

  • Certains modules dépendant de variables de XOOPS2, et ne peuvent pas être installés sur XOOPS Cube Legacy 2.1.
  • Ajouter la possibilité de prévisualiser un bloc personnalisé.
  • Ajouter dans la recherche d'utilisateur la navigation 'suivant' et 'précédant'. Ces options de recherche sont disponibles dans XOOPS2.
  • Parfois IIS n'installe pas les modules avec des fichiers SQL. Si vous rencontrez des problèmes, veuillez nous le signaler.
