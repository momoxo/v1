사용자 모듈 - 공개 블록

로그인 블록
개요

이 블록은 손님에게 인증을 위한 로그인 폼을 표시해 줍니다. 또한 관리설정에서 신규등록을 허가해 둔 경우 신규등록을 위한 register.php 로의 링크도 표시해 줍니다. 그리고 SSL로그인 관련 설정을 한 경우엔 이와 관련된 링크가 표시됩니다.

참고

혹시 실수로 로그인 블록을 제거한 경우엔, user.php 로 직접 액세스해 로그인 작업을 하실 수 있습니다.

온라인상황 블록
개요

이 블록은 현재 사이트에 접속해 이용하고 있는 사용자의 수와 사용자명을 표시해 줍니다.

참고

온라인중인 사용자에 대한 정확한 정보를 얻는 작업은 시스템에 상당한 부하를 주게 됩니다. 따라서 이 온라인상황 블록은 완벽히 정확한 정보를 표시하기 보다는 대략적인 정보를 표시하게 디자인되어 있습니다.

신규등록회원 블록
개요

이 블록은 사이트에 최근 신규등록한 사용자들의 사용자명을 표시해 줍니다.

옵션
x 명을 표시
최근 신규등록자를 표시할 때 몇 명까지 표시할지 인원수 제한을 지정할 수 있습니다.
사용자의 아바타를 표시
예를 선택하시면 사용자의 아바타도 함께 표시되게 됩니다.
투고수랭킹 블록
개요

이 블록은 투고자의 투고수 랭킹을 표시해 줍니다.

옵션
x 명을 표시
몇 명까지 표시할지 인원수 제한을 지정할 수 있습니다.
사용자의 아바타를 표시
예를 선택하시면 사용자의 아바타도 함께 표시되게 됩니다.
다음 특별랭크의 사용자는 제외함
지정한 특별랭크에 해당하는 사용자는 표시하지 않고 제외하게 됩니다.
