Sommaire

  • Ce module de rendu (renderer) vous permet d'utiliser les themes de XOOPS 2.0.x/XOOPS JP 2.0.x.
  • Il n'a aucun mécanisme de remplacement ou de secours. Smarty est utilisé pour afficher des données et gérer le cache.
  • Ce système est d'une vitesse moyenne. Veuillez consulter le fichier package.ini.php pour plus de détails.

Thèmes supplémentaires

  • Les thèmes supplémentaires peuvent être obtenus sur les sites des créateurs de thèmes, les sites de support, etc.
  • Les thèmes distribués sont compressés. Pour utiliser un thème, les fichiers doivent être décompressés avant d'être placés dans le dossier /thèmes de votre site.

Télécharger des Thèmes

  • Le dossier du thème décompressé doit être chargé dans le dossier /themes du répertoire racine de votre XOOPS Cube.
    Le dossier de votre theme doit inclure le fichier theme.html ou packet.ini.php.
    Vous devez télécharger le theme et le décompresser sur votre machine avant de le placer sur votre serveur.
    Mais si la configuration de votre serveur vous permet de décompresser des fichiers, alors chargez le fichier compressé dans le dossier /themes et décompressez le sur votre serveur.

La sélection de thèmes

  • Dans l'option "Gestion des Thèmes" du module 'Legacy System', veuillez cliquez sur le le bouton "Sélectionner" à côté du thème que vous souhaitez utiliser comme le thème implicite ou thème principal de votre site.
  • Chaque thème avec sa case à cocher sélectionnée est affiché dans le bloc de selection de theme.

Désinstaller un theme

  • Ce système gère les thèmes sans utiliser la base de données. Pour effacer un thème, supprimez simplement son dossier du repértoire /themes de votre XOOPS Cube.

Gestion des Templates

  • La liste des ensembles de templates est affichée dans la Gestion des Templates.
  • XOOPS Cube est fournit avec un ensemble de templates par défaut. Lorsque vous installez des modules, leurs templates deviennent des fichiers de cet ensemble de templates.
  • L'ensemble de templates par défaut ne peut pas être effacé ni modifié. Si vous souhaitez modifier les templates pour votre site, vous devez tout d'abotd "cloner" l'ensemble de templates par défaut. Le bouton "cloner" est sur la droite, dans la colonne de contrôle. Une fois que vous avez un nouveau ensemble de templates, vous devez tout d'abord l'activer en cliquant sur le bouton "Sélectionner" dans la colonne de contrôle. Maintenant, les véritables fichiers templates peuvent être édités depuis la "Gestion des Templates" ou "Fichiers Templates".
  • Si vous avez un Ensemble de Templates au format tar.gz, vous pouvez l'envoyer et l'utiliser sur votre site via l'interface administrative. Cliquez simplement sur le lien "Envoyer un Ensemble de Templates" dans la "Gestion des Templates".

Les Fichiers Templates

  • Une liste des fichiers templates est affichée, et chaque fichier peut être édité en cliquant sur l'icône éditer (un crayon) dans la colonne de contrôle.
