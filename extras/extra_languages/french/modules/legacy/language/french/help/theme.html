Les Themes sont l'apparence de votre XOOPS Cube. Vous pouvez changer l'apparence de votre site en changeant le theme principal comme on change d'habits. Changeons l'apparence de votre site. Téléchargez des themes depuis les sites des designers qui ont publiés des themes pour XOOPS Cube.

Comment changer le theme principal

Le theme principal est le theme par défaut de votre site. Vous pouvez spécifier un seul theme principal sur votre site.

  1. Cliquez sur le lien 'Themes' dans le menu du module Legacy.
  2. Cliquez sur le bouton 'Selectionnez' du theme que vous voulez utiliser comme theme principal.

Après valider votre choix, vous pouvez visiter votre site et voir appliqué le nouveau design.

Comment installer de nouveaux themes

Vous pouvez télécharger d'autres themes qui sont distribués sous forme de fichiers compressés. Pour utiliser un theme, placez les fichiers décompressés dans le repértoire /themes de votre installation XOOPS Cube.

Attention au repértoire theme!
  • Le dossier du theme doit avoir un fichier theme.html ou package.ini.php.
Note sur les themes

Un theme peut faire appel a un système-rendu spécial.
Le module 'Legacy Render System' qui est installé par défaut, supporte le format des themes de XOOPS2.

Comment utiliser la Sélection du Theme

Les utilisateurs peuvent sélectionner leur theme favori depuis la liste de themes que vous avez spécifié. Pour cela, vous devez les choisir dans la liste de themes et installer le bloc 'Theme Select'.

  1. Tout d'abord, vous devez créer une liste de themes. Cliquez sur le lien 'Themes' dans le menu du module Legacy.
  2. Spécifiez les themes qui seront sélectionnables. Ensuite, cliquez sur le bouton 'Envoyer'.
  3. Installer le bloc 'Theme Select'. Si vous ne savez pas comment installer des blocs sur XOOPS Cube, lisez l'aide à ce sujet.
