Comment utiliser les Blocs

Les blocs vous permettent d'afficher davantage de contenu avec les modules. Si un module possède des blocs, ils seront ajoutés.

Installer des Blocs

Avant tout, il vous faut les installer pour les utiliser. Cliquez ici pour installer des blocs. Dans le formulaire, vous pouvez changer le nom du bloc et déterminer sa visibilité par groupe d'utilisateurs. Le poids est l'ordre d'affichage dans la colonne que vous spécifiez.

Colonne

La plupart des thèmes ont souvent trois colonnes. La colonne Centre peut avoir à son tour plus de deux colonnes. Ces colonnes sont appelées, la colonne de Gauche, la colonne de Centre-Gauche, la colonne de Centre-Centre, la colonne de Centre-Droit et la colonne de Droite. Vous pouvez afficher le bloc sur la colonne de votre choix avec le module que vous spécifiez.

Quelques thèmes n'ont pas de colonnes de Gauche ou Droite. Dans un tel cas, le contenu d'une colonne pourrait être insérée dans une des autres colonnes. Vous aurez besoin de lire le manuel de ces thèmes pour comprendre leurs caractéristiques.

Note

Un Système de Rendu spécial peut vous apporter des fonctions spéciales pour les blocs. Si vous l'avez installé, lisez le manuel de votre Système de Rendu.

Gestion de Blocs

Vous pouvez changer un bloc, à tout moment, depuis le module Legacy. Cliquez ici. L'utilisation est basée sur les autres fonctions de contrôle. Cliquez l'icône éditer pour changer le bloc.

Désinstaller un Bloc

Si vous voulez retirer un bloc de l'affichage, cliquez sur l'icône désinstaller. Le bloc se retrouve dans la liste de blocs non-installés. Le bloc est gardé jusqu'à ce que vous désinstallez le module auquel il est associé. Vous pouvez réinstaller à tout moment.

POUR LES UTILISATEURS DE XOOPS2

XOOPS Cube Legacy 2.1 sépare la liste de blocs non-utilisés de la gestion de Blocs. Tout comme la gestion de Modules.
