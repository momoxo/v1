테마는 XOOPS Cube 의 스킨(Skin)이라 할 수 있습니다. 여러분들은 테마 변경을 통해 사이트의 외관을 다양하게 꾸미실 수 있습니다. 현재 많은 사이트들이 XOOPS Cube 용 테마를 배포하고 있으니 다운받으셔서 사이트를 다양하게 꾸며보시기 바랍니다.

메인 테마 변경법

메인 테마는 사이트의 디폴트 테마 입니다. 하나의 사이트에 대해 하나의 디폴트 테마를 지정하실 수 있습니다.

  1. 호환모듈(Legacy module)의 사이드메뉴에서 테마관리를 선택해 주세요!.
  2. 메인 테마로 사용하실 테마를 고르신 후 선택 버튼을 클릭하시면 됩니다.

설정을 완료하신 후 사이트에 직접 접속해 보시기 바랍니다. 새로운 테마가 적용된 사이트를 보실 수 있으실 겁니다.

추가 테마의 입수

테마 개발자의 사이트등을 통해 테마를 추가 입수하실 수 있습니다. 테마는 배포시에 보통 압축파일형식으로 배포되니 다운받으신후 압축을 해제하여 XOOPS Cube의 지정위치( 예: /themes )에 업로드하여 사용하시면 됩니다.

테마 디렉토리에 주의!
  • 다운받아 압축을 해제한 테마의 디렉토리를 XOOPS Cube 의 테마용 디렉토리( /themes )에 업로드하시기 바랍니다.
  • 업로드시에는 디렉토리가 이중으로 만들어지지 않게 주의하시기 바랍니다. 예를 들어 default 테마의 경우 /themes/default 디렉토리내에 theme.html 또는 package.ini.php 파일이 존재하여야 합니다.
테마의 주의/설명서를 꼭 읽어보시기 바랍니다.

특별한 테마의 경우 특정 랜더시스템(render-system)을 필요로 하는 경우가 있습니다. XOOPS Cube의 호환 랜더시스템(Legacy Render System)은 XOOPS2 용 테마를 위한 것입니다. (물론 특정 랜더시스템을 추가설치하시면 새로운 포맷의 테마를 사용하실 수 있습니다.) 그러므로 새로운 테마를 설치하실때는 반드시 해당 테마의 주의/설명서를 잘 읽어보신 후 설치하시기 바랍니다.

사용자의 테마 선택

사용자들이 지정된 테마리스트에서 원하는 테마를 선택적으로 사용할 수 있게 할 수 있습니다. 이를 위해선 선택가능 테마를 리스트업, 테마선택 블록 설치작업이 필요합니다.

  1. 먼저 테마리스트를 만듭니다. 호환모듈(Legacy module)의 사이트메뉴에서 테마관리를 선택해 주세요!
  2. 선택가능하게 할 테마를 체크하신 후 보내기 버튼을 클릭하시기 바랍니다.
  3. 테마 선택 블록을 설치합니다. 블록 설치에 대해 잘 모르시는 분들은 블록관련 Help를 참조해 주세요!
