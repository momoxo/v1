Sommaire

  • Ce module permet au site d'éxecuter son propre service de messagerie privé.
  • En outre, un service virtuel concernant la messsagerie privée est fournit pour assurer la compatibilité avec les versions de XOOPS 2.0.x.
  • Les utilisateurs peuvent utiliser la boîte de saisie de texte ou le menu déroulant pour choisir le destinataire du message.

Préférences

  • Par défaut, les utilisateurs peuvent choisir le destinataire du message via le menu déroulant. Cependant, la situation peut se compliquer avec cette option si le site atteint un nombre important d'utilisateurs enregistrés. L'administrateur du site peut alors changer les préférences, de manière à que le destinataire du message soit indiqué par les utilisateurs.
